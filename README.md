# smorgasburgAR
TMG Smorgasburg Project using AR
To Run: 
1. Update table.json by running 
  node app.js
2. Run https by running 
  sudo npx http-server -S -C cert.pem -K key.pem
  
 File Info:
 1. Excel.html
    Displays excel info
 2. AR.html
    Displays AR info parsed from excel
 3. codepen.html
    Displays hardcoded AR info
